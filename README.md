# Crypto-tracker

A web application that allows users to track cryptocurrency prices in real time.

## Features
- View real-time crypto prices
- Search for cryptocurrencies
- View price trends and changes
- Mobile-friendly design

## Tech Stack
- React
- JavaScript
- CSS
- Crypto API (to be added)
